What is Blockcred?

Blockcred is a public, mineable blockchain. It can also be seen as an open-source protocol for the public blockchains, which offers a rich set of features including extensive configurability, rapid deployment, native assets, and data streams.
You can publish your data in key-value pair format on our blockchain.

How is Blockcred different from traditional storage systems?

Since Blockcred is a public blockchain, it consists of several decentralized nodes which verify each and every transaction while adding new blocks into the blockchain. Each node participates in administration: all nodes verify new additions to the blockchain, and are capable of entering new data into the blockchain. Traditional databases allow the modification of data, but with the Blockcred blockchain the data is secured and cannot be tampered with, altered, or modified once published and confirmed in the Blockcred blockchain ledger.

How does Blockcred work?

Blockcred allows the user to upload data (in key-value pair format) onto the blockchain with ease and in a single XBC transaction. Users can upload multiple entries in the form of key-value pairs, but it has to be one pair per transaction. Blockcred is a public blockchain, and it also allows users to retrieve their uploaded data by the use of a key. The cost of data upload is calculated as per the data size and awarded to the miners in the form of XBC.

What are the blockchain specifications for Blockcred?

The Blockcred specifications are as follows:
• Avg. Block Size: 250 bytes
• Max Block Size: 8 MB
• Avg. Block Frequency: 15 seconds
• Mining Algorithm: PoW
• Mining Reward: 10 XRK
• Max Transaction Size: 4 MB
• Max OP_RETURN Transaction Size: 2 MB

Can I mine XBC?

Yes. Anyone can become a miner with Blockcred. You need to download the needed mining software, and then you can start mining. You can follow the mining guide that will be available once we open the mining for the crowed.

What are the minimum H/W requirements for XBC mining?

Anyone with a personal laptop/computer can begin mining for XBC. The minimum system requirements are as follows:
• Linux: 64-bit, supports Ubuntu 12.04+, CentOS 6.2+, Debian 7+, Fedora 15+, RHEL 6.2+.
• Windows: 64-bit, supports Windows 7, 8, 10, Server 2008 or later.
• Mac: 64-bit, supports OS X 10.12 (we hope to support earlier versions soon).
• 1 MB of RAM
• 5 GB of disk space

What is XBC?

XBC is the name (or ticker) of the tokens which are used as an incentive and payment model for issuing a certificate onto the Blockcred blockchain. XBC tokens are used as a gas for issuing a certificate over the Blockcred blockchain.
